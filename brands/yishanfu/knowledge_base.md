# 易善复 (Essentiale)

## 品牌概述

易善复（Essentiale）是源自德国的专业护肝品牌，由赛诺菲（北京）制药有限公司生产，华润三九全权代理。作为中国No.1护肝OTC产品，易善复深耕医院、零售、电商渠道，已入驻中国市场30年（1994年进入中国）。

- **品牌 Slogan**: "易善复，为肝守护"
- **品牌定位**: 天然肝修护专家
- **品牌调性**: 专业、天然、权威、值得信赖
- **品牌视觉**: 主色调为金色/橙色，以蜂巢结构象征磷脂分子层，肝脏金色意象

## 产品信息

### 基本信息
- **商品名称**: 易善复（胶囊）
- **通用名称**: 多烯磷脂酰胆碱胶囊 (Polyene Phosphatidylcholine Capsules)
- **英文名**: Essentiale
- **核心成分**: 多烯磷脂酰胆碱 (PPC)，每粒含228mg
- **药品类别**: 非处方药（OTC甲类，红标）
- **批准文号**: 国药准字H20059010
- **生产企业**: 赛诺菲（北京）制药有限公司
- **原料来源**: 从大豆中高度纯化提取，核心成份德国进口
- **提纯纯度**: 近90%，PPC占原料主成份的94-96%
- **专利技术**: 德国专利高纯提取技术（1粒易善复 ≈ 0.75公斤大豆提取的磷脂精华）

### 产品规格
- 24粒装（零售）
- 36粒装（零售核心产品及36粒plus）
- 96粒装（零售大包装）

### 适应症
1. 辅助改善中毒性肝损伤（如药物、毒物、化学物质和酒精引起的肝损伤等）
2. 辅助改善脂肪肝和肝炎患者的食欲不振、右上腹压迫感

### 用法用量
- **适用人群**: 12岁以上儿童、青少年和成年人
- **初始剂量**: 每日3次，每次2粒（456mg），每日最大不超过6粒（1368mg）
- **维持剂量**: 一段时间后，可减至每日3次，每次1粒（228mg）
- **服用方式**: 需随餐服用，用足够量的液体整粒吞服，不要咀嚼

### 禁忌与注意事项
- 已知对大豆制剂、磷脂酰胆碱过敏和/或对本品中任何成分过敏的患者禁用
- 大剂量服用时偶尔会出现胃肠道紊乱（胃部不适、软便和腹泻）
- 极罕见情况下可能出现过敏反应（皮疹、荨麻疹、瘙痒等）
- 全国城镇医保目录药品（2020版国家医保乙类，医保编号49）

## 作用机制

### 三重作用机制
1. **修复** — 加速受损肝细胞膜修复和再生（必需磷脂特异结合肝细胞膜，增加有丝分裂活性）
2. **保护** — 细胞水平的保护作用（增加膜流动性，保护和增加遗传物质）
3. **更新/解毒** — 抑制脂质过氧化，预防肝脏脂肪变性、纤维化和肝硬化

|------|:---:|
| 抗氧化 | + |
| 转甲基 | + |
| 膜稳定 | + |
| 膜修复 | + (独有) |
| 改善血脂 | + |
| 解毒 | + |

> **独特性**: 多烯磷脂酰胆碱是唯一具有"膜修复"功能的保肝药物，这是其区别于其他保肝药物（甘草酸二胺、水飞蓟素、谷胱甘肽、双环醇等）的核心差异化优势。

## 品牌核心传播概念 (Brand Concept)

### 沟通概念
**"易善复，为肝守护"**

### 人群洞察
不良的生活方式往往会对肝脏造成损伤，进一步导致脂肪肝等更严重的肝脏问题。这类人群想要以更天然、温和的方式修护肝脏，作为他们日常的补充。

### 品牌定位
**天然肝修护专家**

### 差异化优势 (POD)
天然必需磷脂，三步修护肝脏：
1. **辅助改善** — 改善脂肪肝食欲不振等症状
2. **肝脏保护** — 辅助改善酒精、药物等引起的中毒性肝损伤
3. **肝膜再生** — 加速细胞膜的再生

### 信任支撑 (RTB)
- **更温和**: 辅助改善肝损伤症状，纯植物提取温和保护，提供膜保护修复所需的营养
- **更安全**: 11部权威指南推荐，中国上市30年，62年保肝护肝历史
- **更专业**: 用作多种肝病治疗的辅助用药，高浓度多烯磷脂酰胆碱成分，源自德国、德国制造
- **更有效**: 多烯磷脂酰胆碱有效成分，符合OTC的国药标准，200多项临床研究支持

## 临床证据与权威推荐

### 权威指南推荐
已被中国30+份专家共识与临床指南推荐，包括：
- 《NAFLD诊治指南》II级推荐（2010版）
- 《HBV防治指南》II级推荐（2015版）
- 《酒精性肝病诊疗指南》强力推荐（2010版）
- 《肝脏炎症及其防治专家共识》I级推荐（2014版）
- 《非酒精性脂肪性肝病防治指南》（2018版）
- 《中国成人2型糖尿病合并非酒精性脂肪性肝病管理专家共识》（2021版）
- 《抗结核药物性肝损伤诊治指南》（2019版）

### 关键临床研究
1. **EXCEL研究** (2025年，最高证据等级RCT): 3/6个月显著降低肝脏脂肪，比对照组效率高2.5倍；显著改善疲劳症状；改善血糖控制
2. **RELIEF研究** (中国首个MAFLD真实世界研究): 6个月显著降低FIB-4指数（降低肝纤维化风险）；3个月改善肝功和血脂指标
3. **MANPOWER研究** (2019-2020): 改善超声检测结果、肝功能指标、血脂，安全性和依从性良好
4. **Dajani研究** (2015): 改善超声检测结果、减轻临床症状、改善肝功能指标

### 竞品对比优势
- **vs 护肝片**: 临床有效率更高（85.42% vs 62.50%），显著降低肝功能指标
- **vs 水飞蓟宾**: 总治疗有效率更高（92.68% vs 70.73%），不良反应发生率更低（7.32% vs 31.71%）
- **vs 甘草酸二铵**: 改善NASH疗效更佳（86.7% vs 60.0%），血脂改善效果更好
- **vs 双环醇**: 疗效相当但副作用较小

### 临床使用率
多烯磷脂酰胆碱临床使用率居肝病辅助用药首位（22.81%），远超第二名双环醇（17.02%）。

## 目标人群

### 三大核心人群
1. **脂肪肝患者** — 已确诊的脂肪肝群体（36粒核心产品及36粒plus）
2. **慢病共病高风险人群** — 糖尿病、高血压、高血脂等合并脂肪肝风险的患者
3. **不良生活习惯高风险人群** — 熬夜、高脂饮食、过度饮酒的人群（96粒产品及健肝素）

### 零售端目标消费者画像
- **年龄**: 35岁+
- **性别**: 男性为主（肝病患者男女比例5:1）
- **地域**: 一二线城市压力人群为主
- **电商购买人群**: 20-40岁占57%，40-50岁占24%

### 零售端消费者沟通要点

#### 首次购买者
1. 脂肪肝就是病，是中国第一大慢性肝病，不干预易发展为肝纤维化、肝硬化甚至肝癌
2. 70%患者仅改变生活方式无法控制肝损伤，需要药物治疗
3. 易善复三重机制，根源修复受损肝细胞膜，3个月降酶降脂，6个月延缓纤维化进程

#### 竞品转换者
1. 脂肪肝导致肝细胞膜受损，需要从源头修复肝细胞膜
2. 易善复核心成分与肝细胞膜组成相似，是唯一直接修复肝细胞膜的护肝药
3. 24小时直达受损肝脏，3/6个月显著降低肝脏脂肪，纯天然大豆提取

#### 高风险人群（糖尿病/肥胖）
1. 60%糖尿病患者患有脂肪肝，中国肥胖人群脂肪肝患病率高达82%
2. 稳糖减肥护肝要并行
3. 易善复同时改善血糖控制，与二甲双胍联用可进一步改善代谢
