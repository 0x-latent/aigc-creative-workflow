# 共病管理洞察：糖肝共管与心血管代谢性疾病

## 糖尿病与脂肪肝的关联
- 脂肪肝和2型糖尿病互为因果，相互促进
- 我国2型糖尿病患者中，非酒精性脂肪肝的患病率达55.3%
- NAFLD与糖尿病对彼此疾病进程的影响：肥胖/腹部肥胖 → 代谢综合征/胰岛素抵抗 → 糖尿病与NAFLD相互促进 → 心血管疾病
- 糖尿病、肥胖会加速MAFLD进程，并伴随持续的肝细胞损伤

## 肥胖、高血脂与脂肪肝
- 中国肥胖人群中脂肪肝患病率高达82%
- 肥胖约九成患有脂肪肝、约六成患有高血压、约五成以上患有心脏病
- 高脂血症与NAFLD对彼此疾病进程影响：胰岛素抵抗/肥胖 → 脂肪细胞功能异常 → 高脂血症 → NAFLD → NASH → 晚期肝病/T2DM

## 心血管代谢性疾病共管
- CVD（心血管疾病）在MAFLD患者中发生率 >50%
- 脂肪肝患者容易罹患心脑血管疾病
- 脂肪肝增加恶性肿瘤发病风险
- 脂肪肝会降低人体免疫功能、解毒功能

## 易善复在共病管理中的价值
- **EXCEL研究**: 6个月显著降低HbA1c水平（P=0.0069），改善MAFLD患者血糖控制
- **RELIEF研究**: 3个月显著改善LDL-C水平，降低血脂指标
- **联合用药证据**: 在二甲双胍基础上加用PPC，可进一步降低血脂（TG下降0.88 mmol/L）及肝酶（ALT下降3.84 U/L），86%患者获益
- **三联方案**: 二甲双胍+非诺贝特+易善复，可进一步降脂、保肝，还可抑制炎症反应（HMGB1和IL-17显著下降）

## 共病管理沟通要点
1. 脂肪肝不仅仅是肝脏问题，更是全身代谢紊乱的表现
2. 早筛查、早干预，建议通过药物治疗管理合并症
3. 稳糖减肥护肝要并行
4. 易善复在改善肝功能的同时，还能改善血糖控制和血脂水平
